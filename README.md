## 🛠️ Estructura del Repositorio

El proyecto cuenta con los siguientes archivos principales:

* **`app.py`**: Archivo principal donde se ejecuta la aplicación web.
* **`dashboard.py`**: Contiene la lógica y diseño de las visualizaciones del panel de control.
* **`Sidebar.py`**: Gestiona los elementos de la barra lateral de navegación o filtros.
* **`data_models.py`**: Define las estructuras o modelos de los datos.
* **`data_service.py`**: Capa encargada de la conexión, consulta o procesamiento de los datos.
* **`run.py`**: Script auxiliar para iniciar la ejecución del proyecto.
* **`requirements.txt`**: Lista de dependencias y librerías necesarias de Python.
